# 🌡️ Temperature Converter

A modern, responsive temperature converter web app built with HTML, CSS, and JavaScript. It allows users to convert temperatures between Celsius, Fahrenheit, and Kelvin with a clean and visually appealing interface.

## Live Demo

[Visit site](https://vee-mukoma.github.io/temperature-converter/)

## 🚀 Features

- Convert between:
  - Celsius
  - Fahrenheit
  - Kelvin
- Instant conversion on button click
- Input validation for empty/invalid values
- Fully responsive and modern UI
- Smooth animations and hover effects
- Clean gradient-themed design

## 🎨 UI Highlights

- Dark-themed modern interface
- Custom CSS variables for easy theming
- Gradient text and buttons
- Animated header accent line
- Responsive card layout
- Smooth hover and focus effects

## 🛠️ Built With

- HTML5
- CSS3 (Flexbox, Variables, Animations)
- JavaScript (Vanilla JS)

## ⚙️ How It Works

1. User enters a temperature value
2. Selects the unit to convert **from**
3. Selects the unit to convert **to**
4. Clicks the **Convert** button
5. JavaScript:
   - Validates input
   - Determines conversion logic using `switch`
   - Applies correct formula
   - Displays formatted result
